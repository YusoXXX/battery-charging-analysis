# Method Notes

## Scope

This document summarizes the methodological background of the battery charging analysis scripts.

## Current focus

The current repository focuses on:

- Δt(Q)-based charging comparison
- CSV-based battery test data processing
- reproducible analysis of charging behaviour

## Input data

The scripts are designed for data exported from:

- Rohde & Schwarz NGU201
- Sefram DAS60

## Notes

This repository is under structured development and will be expanded with additional scripts for battery data analysis.
